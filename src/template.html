Hello, I am Woon Zhen Fei, a web developer.

I have past working experience as an Operation Executive and Service Technician in different field. During these jobs, I realized I would really like to have a job that spends a lot of time in front of a computer and so began my journey to learn web development.

I have prior knowledge about programming (C programming and MATLAB) during my university years. Since I started my journey in web development, I have successfully completed CS50: Introduction to Computer Science, a Harvard online course. I am currently learning through The Odin Project's Full Stack JavaScript path.

This Homepage is actually a project of the final lesson in TOP's Advanced HTML and CSS course. I did not really follow the design template provided as I want to make it unique and use it as my first actual portfolio.

I am learning React and looking for my first job in web development.
